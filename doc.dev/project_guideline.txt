1. SYSTEM OVERVIEW DOCUMENT
1.1 System Name
Inclusive Growth Portal (IGP)
1.2 Purpose
The Inclusive Growth Portal is a secure internal web-based platform designed to:
•	Manage organizational documents
•	Provide employee telephone directory
•	Control access based on user roles
•	Improve communication and information sharing
1.3 Target Users
•	Admin
•	HR Staff
•	Finance Staff
•	Program Team
•	IT Team
•	Management
•	General Employees
2. SYSTEM ARCHITECTURE (High-Level)
Recommended Architecture:
•	Web-based system
•	Centralized database
•	Role-Based Access Control (RBAC)
•	Cloud or On-Premises hosting
SYSTEM MODULES STRUCTURE
MODULE 1: Authentication System
Features:
•	Login page
•	Logout
•	Password reset
•	Role-based access
User Roles:
Role	Permissions
Super Admin	Full system control
Admin	Manage users & documents
Department Head	Upload/manage department documents
Staff	View/download allowed documents
Guest	Limited access

MODULE 2: Dashboard (Home Page)

•	Welcome message

•	Statistics (Total documents, users, etc.)

Our logo uses two carefully selected colors to embody our values:

•	Dark Green (CMYK 82, 28, 100, and 15): Represents growth, inclusion, sustainability, nature, empowerment, and foundational sectors such as education, health, and technology.
•	Golden Yellow (CMYK 2,25,93,0): Symbolizes hope, creativity, energy, innovation, and optimism—qualities that drive our initiatives and inspire the communities we serve.

N.B: Together, these colors and visual elements convey a message of inclusive progress, Inclusive Development, mutual respect, and limitless potential.
